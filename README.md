# RandomAnimePicker

Pick a random anime from MyAnimeList using jikan api

## Dependencies

* Node >= 18
* Yarn

Install dependencies
```sh
$ yarn install
```

## Build

Build on development mode
```sh
$ yarn dev
```

Build on production mode

* SSR (Server side rendering)

```sh
$ yarn build
```
* SSG (Static site generation)

```sh
$ yarn generate
```
